# Hospital Management System

## Project Overview

The Hospital Management System is a MySQL-based database project designed to manage hospital operations including patients, doctors, departments, appointments, treatments, rooms, and billing. The project demonstrates database design, data management, and advanced SQL concepts through real-world healthcare scenarios.

## Features

* Patient Management
* Doctor and Department Management
* Appointment Scheduling
* Treatment Tracking
* Billing and Payment Management
* Room Allocation
* Revenue and Operational Reporting

## SQL Concepts Used

* Joins
* Aggregate Functions
* Subqueries
* Common Table Expressions (CTEs)
* Window Functions (RANK, DENSE_RANK, ROW_NUMBER)
* Stored Procedures
* Triggers
* Indexes
* Transactions (COMMIT, ROLLBACK, SAVEPOINT)

## Database Modules

* Patients
* Doctors
* Departments
* Appointments
* Treatments
* Rooms
* Bills
* Payments

## Project Highlights

* Designed and implemented a relational database with 15 interconnected tables.
* Managed over 30,000+ records using CSV datasets.
* Developed advanced SQL queries for reporting and analytics.
* Implemented Stored Procedures and Triggers to automate database operations.
* Used CTEs and Window Functions for ranking and business insights.
